Detailed Description

A Board where monsters and player runs.

Class represents single level loaded from file where player can play on.

It basically is a 2D board of tiles (chars), where each can be wall, invisible wall (around blank spaces, so it looks prettier :D), blank space, food or star (to slow down ghosts and get them into frightened mode).

Also contains some additional info, like starting positions for player, ghosts and number of total / eaten dots.
